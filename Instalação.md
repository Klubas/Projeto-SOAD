# SOAD - Sistema organizacional de armazenagem e distribuição

## Manual para instação

### Dependências

* Python3.7 
* Qt5
* PySide2
* Pydal
* psycopg2
* shiboken2

### Downloads

* Python

####### No instalador marque a opção para adicionar o python ao 'Path' do Windows

* Qt5
